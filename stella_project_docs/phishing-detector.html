Phishing Detector

Project Overview

Phishing Detector is a lightweight triage tool designed to automate the analysis of suspicious emails and speed up the SOC response process. The tool breaks down incoming emails and applies threat intelligence checks, content analysis, and scoring to assist analysts in making rapid decisions.

Workflow Example

  1. Analyst forwards suspicious email to tool
  2. Tool parses email, extracts URLs and attachments
  3. URLs are scanned; attachments are hashed and analyzed
  4. Tool returns a verdict and threat score in seconds
